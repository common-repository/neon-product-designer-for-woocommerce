<h4>Overview</h4>

The best WooCommerce product customizer for neon signs. 

NPD allows you to extend your business of personalization of neon signs by offering you a configurator to allow your customers to customize online prior to order their custom neon sign. Easy and quick to set up.

== Installation ==

<div class="block-content"><h4>Minimum Requirements</h4>

<ul>
<li>WordPress 4.0 or greater</li>
<li>Woocommerce version 3.0 or greater</li>
<li>PHP version 5.2.4 or greater</li>
<li>MySQL version 5.0 or greater</li>
</ul>

<p><strong>This section describes how to install the plugin and get it working</strong></p>

<h4>Automatic installation (easiest way)</h4>

<p>To do an automatic install of Neon Product Designer, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.</p>

<p>In the search field type "Neon Product Designer" and click Search Plugins. Once you have found it you can install it by simply clicking "Install Now".</p>

<h4>Manual installation</h4>

<p><strong>Uploading in WordPress Dashboard</strong></p>

<ol>
<li>Download <code>neon-product-designer.zip</code></li>
<li>Navigate to the 'Add New' in the plugins dashboard</li>
<li>Navigate to the 'Upload' area</li>
<li>Select <code>neon-product-designer.zip</code> from your computer</li>
<li>Click 'Install Now'</li>
<li>Activate the plugin in the Plugin dashboard</li>
</ol>

<p><strong>Using FTP</strong></p>

<ol>
<li>Download <code>neon-product-designer.zip</code></li>
<li>Extract the <code>neon-product-designer</code> directory to your computer</li>
<li>Upload the <code>neon-product-designer</code> directory to the <code>/wp-content/plugins/</code> directory</li>
<li>Activate the plugin in the Plugin dashboard</li>
</ol>

<p>The WordPress codex contains <a href="http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation" rel="nofollow">instructions on how to install a WordPress plugin</a>.</p>

<h4>Updating</h4>

<p>You can use automatic update to update the plugin safely.</p></div>
